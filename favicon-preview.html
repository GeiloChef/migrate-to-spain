Favicon Preview

This page shows your favicon in different sizes. You can right-click and save these images to convert them to ICO format.

16x16
32x32
48x48

Instructions:

  1. Start your Nuxt dev server: npm run dev
  2. Open this file in your browser: file:///path/to/favicon-preview.html
  3. Right-click on the 32x32 image and save it
  4. Use an online converter to convert the saved image to ICO format
  5. Replace the existing public/favicon.ico with the converted file
